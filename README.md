# Task for FE_Intern
This is the task given by the company to complete within time for a position of Frontend Engineering Intern .

# Task Given :
### The task requires the development of a "Session Timer Widget" frontend component. 
This
component must incorporate a Start button, a Stop button, and a digital timer display showing
the elapsed time in minutes and seconds (mm:ss). A mandatory feature is the automatic
cessation of the timer upon reaching a duration of 30 minutes. The assessment criteria for this
task are defined as "Precision" and "Performance." The use of artificial intelligence tools is
permitted, and the execution must involve live screen sharing.

# Title: Session Timer Widget
## Testing: Precision, Performance
## Task: A clean component with:
-  Start button
- Timer counter (mm:ss)
- Stop button
- Auto-stop at 30 mins
